CJCONF-REMOVE(1)
================
:doctype: manpage


NAME
----
cjconf-remove - Deletes your config file entirely.


SYNOPSIS
--------
*cjconf-remove*


DESCRIPTION
-----------
cjconf-remove(1) is an alias for "rm /etc/cjdroute.conf".

It doesn't take any arguments.


EXAMPLES
--------
*cjconf-remove*::
    Permanently deletes the config file at /etc/cjdroute.conf if it exists.
